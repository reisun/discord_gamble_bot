## 設計メモ: 複数選択賭けルール

ゲームごとに賭け方式を設定できる。

| 方式 | 識別子 | 重複 | 説明 |
|------|--------|------|------|
| 単数 | `single` | - | 記号1つを選ぶ（デフォルト） |
| 複数-選択一致 | `multi_unordered` | 不可 | N個を順不同で選択、すべて一致で当選 |
| 複数-順番一致（重複なし） | `multi_ordered` | 不可 | N個を順番通りに選択、記号と順番が一致で当選 |
| 複数-順番一致（重複あり） | `multi_ordered_dup` | 可 | 同じ記号を複数回選択可。記号と順番が一致で当選 |

`multi_unordered` は API サーバー側で昇順ソートして正規化する（Bot 側でもソートするが API 側でも保証）。  
倍率計算は `bets.selected_symbols` の組み合わせ文字列単位のパリミュチュエル方式（`single` と統一ロジック）。
