## 共通レイアウト

```
┌─────────────────────────────────────────────────────────────┐
│  ヘッダー: [ロゴ]                             [権限バッジ]  │
├──────────────────┬──────────────────────────────────────────┤
│  左サイドバー    │  メインエリア                           │
│                  │  パンくずリスト                         │
│  - イベント一覧  │  ─────────────────────────────────────  │
│  - ユーザー管理  │                                         │
│  - ポイント管理  │  (各画面のコンテンツ)                   │
│                  │                                         │
└──────────────────┴──────────────────────────────────────────┘
```

### ルーティング方式
GitHub Pages でホスティングするため、**ハッシュルーティング**（Hash Routing）を採用する。  
すべての画面 URL は `#` 以降で表現され、サーバーへのリクエストは発生しない。

```
https://<github-pages-url>/#/events
https://<github-pages-url>/#/events/1/edit
```

### 権限
| 権限 | 条件 |
|------|------|
| 管理者 | ハッシュ内クエリパラメータ `?token=<正しいトークン>` |
| 一般ユーザー | それ以外 |

トークンはハッシュフラグメント内に含めることでサーバーのアクセスログに残らない。  
例: `https://<github-pages-url>/#/events?token=xxxxxxxxxxxxxxxx`

管理者専用要素は権限なし時に非表示とする。

## W-04: ゲーム編集（作成・編集兼用）

**パンくず（新規）**: ホーム > イベント一覧 > [イベント名] > ゲーム一覧 > 新規作成  
**パンくず（編集）**: ホーム > ... > ゲーム一覧 > [ゲームタイトル] > 編集

### 画面構成
```
┌──────────────────────────────────────────────────────┐
│  ゲームタイトル  [________________________]          │
│  説明            [________________________]          │
│                  [________________________]          │
│  締め切り日時    [  日付  ] [  時刻  ]               │
│                                                      │
│  賭け方式        [単数 ▼]                            │
│                  ℹ 記号を1つ選ぶ通常の賭け方式です。  │
│                  ※方式が「単数」以外の場合のみ表示:  │
│  選択数          [__] 個                             │
│                                                      │
│  賭け項目                                            │
│  ┌────────────────────────────────────────────────┐  │
│  │  記号       項目名                              │  │
│  │  [_____]    [_______________________]  [ 削除 ] │  │
│  │  [_____]    [_______________________]  [ 削除 ] │  │
│  │  [ + 項目追加 ]                                │  │
│  └────────────────────────────────────────────────┘  │
│                                                      │
│              [ キャンセル ]  [ 保存 ]                │
└──────────────────────────────────────────────────────┘
```

### フォーム項目

| フィールド | 種類 | 必須 | バリデーション |
|------------|------|------|---------------|
| ゲームタイトル | テキスト | ○ | 1〜100文字 |
| 説明 | テキストエリア | - | 最大500文字 |
| 締め切り日時 | 日時ピッカー | ○ | 現在時刻より未来 |
| 賭け方式 | セレクトボックス | ○ | `単数` / `複数-選択一致` / `複数-順番一致（重複なし）` / `複数-順番一致（重複あり）`（デフォルト: `単数`）。選択中の方式の説明文を直下に表示する |
| 選択数 | 数値 | 条件付き | 賭け方式が「単数」以外の場合のみ表示・必須。2以上の整数 |
| 賭け項目 > 記号 | テキスト | ○ | 半角英字(A〜Z)または半角数字(1〜9)、同一ゲーム内で一意 |
| 賭け項目 > 項目名 | テキスト | ○ | 1〜50文字 |

賭け項目は2行以上必須。記号はユーザーが Bot コマンドで賭ける際の短縮指定子として使用する。  
複数方式の場合、賭け項目数が「選択数」以上であること（「選択数 ≤ 賭け項目数」でないと保存不可）。

#### 公開済みゲームの編集制限

ゲームが公開済み（`is_published = true`）の場合、編集できるフィールドは以下のみ。それ以外のフィールドは読み取り専用（グレーアウト）で表示する。

| フィールド | 公開前 | 公開後 |
|-----------|--------|--------|
| ゲームタイトル | 編集可 | 編集可 |
| 説明 | 編集可 | 編集可 |
| 締め切り日時 | 編集可 | **読み取り専用** |
| 賭け方式 | 編集可 | **読み取り専用** |
| 選択数 | 編集可 | **読み取り専用** |
| 賭け項目 > 記号 | 編集可 | **読み取り専用** |
| 賭け項目 > 項目名 | 編集可 | 編集可 |
| 賭け項目の追加・削除 | 可 | **不可** |

#### 賭け方式の説明文（セレクトボックス直下に表示）

| 方式 | 表示する説明文 |
|------|--------------|
| 単数 | 記号を1つ選ぶ賭け方式です。 |
| 複数-選択一致 | 複数の記号を選び、順序に関わらず選択した記号がすべて一致すれば当選する賭け方式です。 |
| 複数-順番一致（重複なし） | 複数の記号を順番通りに選び、記号と順番が完全に一致すれば当選する賭け方式です。同じ記号は1回しか選べません。 |
| 複数-順番一致（重複あり） | 複数の記号を順番通りに選び、記号と順番が完全に一致すれば当選する賭け方式です。同じ記号を複数回選べます。 |

## W-05: ゲーム状況

**パンくず**: ホーム > ... > ゲーム一覧 > [ゲームタイトル] > 状況

### 画面構成

**管理者向け表示（例: 複数-順番一致 3連）**
```
ゲームタイトル: 第1試合  [賭け方式: 複数-順番一致（重複なし） / 選択数: 3]
締め切り: 2024-01-01 12:00  (残り XX分)
状態: 受付中 / 締め切り済 / 結果確定

── 賭け組み合わせ・倍率（パリミュチュエル方式） ─────────
┌──────────────────────────────────────────────────────────┐
│  BDE: チームB→チームD→チームE  ×1.67倍  (600pt / 3人) │
│  BED: チームB→チームE→チームD  ×2.50倍  (400pt / 2人) │
└──────────────────────────────────────────────────────────┘

── ゲーム結果 ──────────────────────────────────────────────
  確定済みの場合: 「当選: BDE（チームB→チームD→チームE）」と表示
  未確定の場合: 「結果未確定」と表示

結果を確定する（締め切り後に有効化）:
  [単数]       当選項目を選択: [ A: チームA ▼ ]
  [複数-選択一致]  当選項目を選択（順不同）:
                   [ ] A: チームA  [ ] B: チームB  [✓] D: チームD  [✓] E: チームE  ...
                   ※ 選択数: 3 に達したら「確定」ボタンが有効化
  [複数-順番一致] 1位: [ A ▼ ]  2位: [ B ▼ ]  3位: [ D ▼ ]  （選択数分のドロップダウン）

[ 確定 ]
```

**一般ユーザー向け表示（結果確定前）**
```
ゲームタイトル: 第1試合
締め切り: 2024-01-01 12:00  (残り XX分)
状態: 受付中 / 締め切り済

── 賭け組み合わせ・倍率（パリミュチュエル方式） ─────────
┌──────────────────────────────────────────────────────────┐
│  BDE: チームB→チームD→チームE  ×--倍                    │
│  BED: チームB→チームE→チームD  ×--倍                    │
└──────────────────────────────────────────────────────────┘
  ※ 倍率・賭け総数・参加人数は結果確定後に公開されます

── ゲーム結果 ──────────────────────────────────────────────
  「結果未確定」と表示
```

**一般ユーザー向け表示（結果確定後）**
```
── 賭け組み合わせ・倍率（パリミュチュエル方式） ─────────
┌──────────────────────────────────────────────────────────┐
│  BDE: チームB→チームD→チームE  ×1.67倍  (600pt / 3人) │
│  BED: チームB→チームE→チームD  ×2.50倍  (400pt / 2人) │
└──────────────────────────────────────────────────────────┘

── ゲーム結果 ──────────────────────────────────────────────
  「当選: BDE（チームB→チームD→チームE）」と表示
```

### 組み合わせ表示フォーマット

| 方式 | 組み合わせ表示例 |
|------|----------------|
| `single` | `A: チームA` |
| `multi_unordered` | `BDE: チームB、チームD、チームE` |
| `multi_ordered` / `multi_ordered_dup` | `BDE: チームB→チームD→チームE` |

### 要素詳細

| 要素 | 表示条件 | 説明 |
|------|----------|------|
| 賭け方式・選択数バッジ | 全員 | ゲームタイトル横に表示 |
| 残り時間 | 締め切り前 | カウントダウン表示 |
| 倍率 | 管理者: 常時 / 一般: 結果確定後のみ | 組み合わせ単位のパリミュチュエル方式で算出 |
| 賭け総数・参加人数 | 管理者: 常時 / 一般: 結果確定後のみ | - |
| 結果確定フォーム | 管理者のみ | 締め切り後に有効化。方式に応じたUI（ドロップダウン/チェックボックス） |
| 結果列 | 全員 | 結果確定後のみ表示 |

### 倍率計算式（パリミュチュエル方式）
```
組み合わせの倍率 = 総賭けポイント合計 ÷ その組み合わせへの賭けポイント合計
当選時獲得ポイント = 賭けたポイント × 倍率（小数点以下切り捨て）
```
賭けが集まるほど倍率が下がる。倍率は賭けが入るたびにリアルタイムで変動する。

## W-06: ユーザー結果一覧

**パンくず**: ホーム > イベント一覧 > [イベント名] > ユーザー結果一覧

**アクセス権限**: 管理者のみ表示可能。ただし管理者が「一般公開」を有効にした場合、一般ユーザーも閲覧可能。

### 画面構成
```
イベント: 〇〇大会

[☑ 一般ユーザーに公開する]  (管理者のみ操作可能)

── ポイントランキング ────────────────────────────────────────────
並び替え: [所持ポイント順 ▼] / [総資産順]  (切り替えボタン)

┌────────────────────────────────────────────────────────────────────────────────┐
│ 順位 │ ユーザー名 │ 最終ポイント │ 増減     │ 勝/敗   │ 借金総額 │ 総資産  │ 総資産増減 │
├────────────────────────────────────────────────────────────────────────────────┤
│  1   │ User A     │  10,500pt    │ +500pt ▲ │  3勝1敗 │     0pt  │ 10,500pt│  +500pt ▲ │
│  2   │ User B     │  10,200pt    │ +200pt ▲ │  2勝2敗 │  3,000pt │  7,200pt│ -2,800pt ▼│
│  3   │ User C     │   9,800pt    │ -200pt ▼ │  1勝3敗 │  5,000pt │  4,800pt│ -5,200pt ▼│
└────────────────────────────────────────────────────────────────────────────────┘

── ゲーム別ポイント推移 ─────────────────────────────────────────
┌─────────────────────────────────────────────────────────────────────┐
│              │   第1試合   │   第2試合   │   第3試合   │   第4試合   │  合計  │
├─────────────────────────────────────────────────────────────────────┤
│  User A      │  +100pt     │   -50pt     │  +200pt     │  +250pt     │ +500pt │
│              │    債:  0pt │    債:  0pt │    債:  0pt │    債:  0pt │ 債: 0pt│
├─────────────────────────────────────────────────────────────────────┤
│  User B      │   +50pt     │  +100pt     │   -30pt     │   +80pt     │ +200pt │
│              │    債: 30pt │    債:  0pt │    債: 50pt │    債:  0pt │ 債:80pt│
├─────────────────────────────────────────────────────────────────────┤
│  User C      │  -100pt     │   -50pt     │   -30pt     │   -20pt     │ -200pt │
│              │    債:100pt │    債: 50pt │    債: 30pt │    債: 20pt │債:200pt│
└─────────────────────────────────────────────────────────────────────┘
  各セル上段: ポイント増減（賭けなし時は `-`）
  各セル下段: 債 = その試合で増加した借金額（借金賭けなし時は `0pt`）
```

### 要素詳細

| 要素 | 説明 |
|------|------|
| 一般公開チェックボックス | 管理者のみ操作。チェック時は一般ユーザーも本画面を閲覧可能 |
| 並び替え切り替え | 「所持ポイント順」と「総資産順」をボタンで切り替え |
| 最終ポイント | イベント中のポイント増減合計（初期ポイントからの変動） |
| ポイント増減 | プラスは ▲ 緑、マイナスは ▼ 赤で表示 |
| 勝/敗 | イベント内の当選/落選ゲーム数 |
| 借金総額 | イベント内の借金総額（`SUM(debt_history.change_amount) WHERE user_id=? AND event_id=?`） |
| 総資産 | `最終ポイント - 借金総額` で算出 |
| 総資産増減 | `総資産 - 初期ポイント` で算出 |
| ゲーム別推移（上段） | 各ゲームのポイント増減（賭けなし時は `-`） |
| ゲーム別推移（下段・債） | そのゲームで借りた借金額（借金賭けなし時は `0pt`） |
