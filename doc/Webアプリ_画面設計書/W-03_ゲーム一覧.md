# W-03: イベント画面

**ハッシュパス**: `#/events/:eventId/games`  
**パンくず**: ホーム > [イベント名]

## 画面構成
```
イベント: 〇〇大会（開催中）

[ ユーザー結果一覧 ]  (管理者のみ表示)      [ + 新規ゲーム作成 ]  (管理者のみ表示)

┌──────────────────────────────────────────────────────┐
│  # │ ゲームタイトル  │ 締め切り       │ 公開 │ 操作  │
├──────────────────────────────────────────────────────┤
│  1 │ 第1試合          │ 2024-01-01 12:00│ 公開  │ [詳細] [公開切替] │
│  2 │ 第2試合          │ 2024-01-01 13:00│ 非公開│ [詳細] [公開切替] │
│ ...│                  │                │       │       │
└──────────────────────────────────────────────────────┘
```

## 要素詳細

| 要素 | 表示条件 | 動作 |
|------|----------|------|
| `ユーザー結果一覧` ボタン | 管理者のみ | W-06 へ遷移 |
| `+ 新規ゲーム作成` ボタン | 管理者のみ | W-04（新規）へ遷移 |
| `詳細` ボタン | 全員 | W-05 へ遷移 |
| `公開切替` ボタン | 管理者のみ | 公開・非公開を切り替え |

> 編集・削除は W-05（ゲーム画面）のタイトル横に配置。
