# W-01: ホーム画面

**ハッシュパス**: `#/events`  
**パンくず**: ホーム

## 画面構成
```
[ + 新規イベント作成 ]  (管理者のみ表示)

┌──────────────────────────────────────────────────────┐
│  イベント名          開催状態     操作               │
├──────────────────────────────────────────────────────┤
│  〇〇大会            開催中 ●    [詳細] [開催中切替] │
│  △△カップ           終了         [詳細] [開催中切替] │
│  ...                                                  │
└──────────────────────────────────────────────────────┘
```

## 要素詳細

| 要素 | 表示条件 | 動作 |
|------|----------|------|
| `+ 新規イベント作成` ボタン | 管理者のみ | W-02（新規）へ遷移 |
| `詳細` ボタン | 全員 | W-03 へ遷移 |
| `開催中切替` ボタン | 管理者のみ | 開催中イベントを切り替え（開催0件もあり得る） |
| 開催状態バッジ | 全員 | 開催中は `●` などで視覚的に区別 |

> 編集・削除は W-03（イベント画面）のタイトル横に配置。
