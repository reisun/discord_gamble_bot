# W-05: ゲーム画面

**ハッシュパス**: `#/games/:id/status`  
**パンくず**: ホーム > [イベント名] > [ゲームタイトル]

## 画面構成

**管理者向け表示（例: 複数-順番一致 3連）**
```
第1試合  [編集] [削除]  [賭け方式: 複数-順番一致（重複なし） / 選択数: 3]
締め切り: 2024-01-01 12:00  (残り XX分)
状態: 受付中 / 締め切り済 / 結果確定

── 賭け組み合わせ・倍率（パリミュチュエル方式） ─────────
┌──────────────────────────────────────────────────────────┐
│  BDE: チームB→チームD→チームE  ×1.67倍  (600pt / 3人) │
│  BED: チームB→チームE→チームD  ×2.50倍  (400pt / 2人) │
└──────────────────────────────────────────────────────────┘

── ゲーム結果 ──────────────────────────────────────────────
  確定済みの場合: 「当選: BDE（チームB→チームD→チームE）」と表示
  未確定の場合: 「結果未確定」と表示

結果を確定する（締め切り後に有効化）:
  [単数]       当選項目を選択: [ A: チームA ▼ ]
  [複数-選択一致]  当選項目を選択（順不同）:
                   [ ] A: チームA  [ ] B: チームB  [✓] D: チームD  [✓] E: チームE  ...
                   ※ 選択数: 3 に達したら「確定」ボタンが有効化
  [複数-順番一致] 1位: [ A ▼ ]  2位: [ B ▼ ]  3位: [ D ▼ ]  （選択数分のドロップダウン）

[ 確定 ]
```

## 組み合わせ表示フォーマット

| 方式 | 組み合わせ表示例 |
|------|----------------|
| `single` | `A: チームA` |
| `multi_unordered` | `BDE: チームB、チームD、チームE` |
| `multi_ordered` / `multi_ordered_dup` | `BDE: チームB→チームD→チームE` |

## 要素詳細

| 要素 | 表示条件 | 説明 |
|------|----------|------|
| `編集` ボタン | 管理者のみ | ゲームタイトル横。W-04（編集）へ遷移 |
| `削除` ボタン | 管理者のみ | ゲームタイトル横。確認ダイアログ後削除し W-03 へ遷移 |
| 賭け方式・選択数バッジ | 全員 | ゲームタイトル横に表示 |
| 残り時間 | 締め切り前 | カウントダウン表示 |
| 倍率 | 管理者: 常時 / 一般: 結果確定後のみ | 組み合わせ単位のパリミュチュエル方式で算出 |
| 賭け総数・参加人数 | 管理者: 常時 / 一般: 結果確定後のみ | - |
| 結果確定フォーム | 管理者のみ | 締め切り後に有効化。方式に応じたUI（ドロップダウン/チェックボックス） |
| 結果列 | 全員 | 結果確定後のみ表示 |

## 倍率計算式（パリミュチュエル方式）
```
組み合わせの倍率 = 総賭けポイント合計 ÷ その組み合わせへの賭けポイント合計
当選時獲得ポイント = 賭けたポイント × 倍率（小数点以下切り捨て）
```
賭けが集まるほど倍率が下がる。倍率は賭けが入るたびにリアルタイムで変動する。
