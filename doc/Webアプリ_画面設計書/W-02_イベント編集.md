# W-02: イベント編集（作成・編集兼用）

**ハッシュパス**: `#/events/new` / `#/events/:id/edit`  
**パンくず（新規）**: ホーム > 新規作成  
**パンくず（編集）**: ホーム > [イベント名] > 編集

## 画面構成
```
┌──────────────────────────────────────────────────────┐
│  イベント名      [________________________]          │
│  初期ポイント    [__________] pt                     │
│                                                      │
│              [ キャンセル ]  [ 保存 ]                │
└──────────────────────────────────────────────────────┘
```

## フォーム項目

| フィールド | 種類 | 必須 | バリデーション |
|------------|------|------|---------------|
| イベント名 | テキスト | ○ | 1〜100文字 |
| 初期ポイント | 数値 | ○ | 1以上の整数（デフォルト: 10000） |
