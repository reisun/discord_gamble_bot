# Webアプリ 共通レイアウト

```
┌─────────────────────────────────────────────────────────────┐
│  ヘッダー: [ロゴ]                             [権限バッジ]  │
├─────────────────────────────────────────────────────────────┤
│  メインエリア                                               │
│  パンくずリスト                                             │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  (各画面のコンテンツ)                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## ルーティング方式
GitHub Pages でホスティングするため、**ハッシュルーティング**（Hash Routing）を採用する。  
すべての画面 URL は `#` 以降で表現され、サーバーへのリクエストは発生しない。

```
https://<github-pages-url>/#/events/<guildId>
https://<github-pages-url>/#/events/<guildId>/1/edit
```

## 権限
| 権限 | 条件 |
|------|------|
| 管理者 | ハッシュ内クエリパラメータ `?token=<正しいトークン>` |
| 一般ユーザー | それ以外 |

トークンはハッシュフラグメント内に含めることでサーバーのアクセスログに残らない。  
例: `https://<github-pages-url>/#/events/<guildId>?token=xxxxxxxxxxxxxxxx`

管理者専用要素は権限なし時に非表示とする。
